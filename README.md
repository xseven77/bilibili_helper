## 🔧 技术实现

### 视频下载
- 通过B站播放页获取视频真实地址
- 支持 FLV/MP4 格式
- 使用浏览器下载功能

### 封面下载
- 获取视频/专栏封面图
- 高清原图下载

### 下载记录
- 使用 Chrome Storage API 本地存储
- 记录：标题、URL、下载时间、类型

## 🚀 使用方法

1. 打开 `chrome://extensions/`
2. 开启**开发者模式**
3. 加载本文件夹
4. 访问B站视频页面，使用插件功能
